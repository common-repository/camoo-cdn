Camoo.Hosting Automatic Integration with CDN for WordPress websites.

== Description ==
Camoo.Hosting Automatic Integration with CDN for websites. This plugin allows you to easily use CDN on your WordPress and accelerate website loading in any country, handling any load on your service. For optimal performance, CAMOO CDN requires WP Super Cache to be installed and activated on your WordPress site.

= Features =
* Replace static links with CDN links

== Frequently Asked Questions ==
= Is WP Super Cache required for CAMOO CDN? =
Yes, CAMOO CDN requires WP Super Cache to be installed and activated. Our plugin works alongside WP Super Cache to ensure your content is delivered efficiently through our CDN.

= What is CAMOO CDN? =
CAMOO CDN is an Automatic CDN Integration for WordPress sites hosted by Camoo.Hosting

= Do I need to do something after the plugin has been activated? =
No, You don't! After you purchase a WordPress Hosting order, our system will add the camoo cdn url automatically in your Managed WordPress installation.

= How can I purchase a WordPress Hosting order? =
All you need is just to [visit our hosting packages](https://www.camoo.hosting/wordpress-hosting).

== Installation ==
1. Install CAMOO CDN from the WordPress Plugin Directory.
2. Ensure WP Super Cache is installed and activated on your WordPress site for CAMOO CDN to function correctly.
3. Activate CAMOO CDN through the 'Plugins' menu in WordPress.
